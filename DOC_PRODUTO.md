# Validador IFC RDC

## Problema
Softwares BIM validam visualmente, mas não explicam erros nem orientam correção.

## Solução
Pipeline IFC textual determinístico que identifica, classifica e orienta correção
de erros de schema, referência e prática BIM.

## Público-alvo
- Engenheiros civis
- Coordenadores BIM
- Escritórios de projeto
- CI de projetos industriais

## Diferencial
Relatório humano + exit codes para automação.
